# 简介

简易的文档内容抽取工具, 支持的文档类型: 
- .txt

逐行抽取纯文本内容;

- .doc(Word 6 - Word 95)

按照文档顺序抽取文本和图片内容;

- .docx(Word 2007)

按照文档顺序抽取文本、图片及表格文本内容;

- .pdf

按页抽取文本和图片内容, 注意页内文本和图片无序;

- .ppt

按页抽取文本和图片内容, 页内按读取顺序显示;

- .pptx

按页抽取文本和图片内容, 页内按照读取顺序显示;
